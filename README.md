# nicotine-website
An interactive site that zooms from a full brain view into a close-up,
with clickable hotspots on brain regions that show informational blurbs.
## Tech
Plain HTML, CSS, and JavaScript — no build tools or frameworks.
## How to run locally
Just open `index.html' in a browser. No installation needed.
live link is https://bmj-project-summer-2026.github.io/nicotine-website/
